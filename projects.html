Full-Stack Hotel Booking System (Completed)

Node.js, PostgreSQL, React, CSS | Mar 2025

This group project involved building a full-stack hotel booking system. Our team of three designed an ER diagram, translated it into a relational schema, and implemented it using PostgreSQL with pgAdmin 4. We ensured data accuracy for hotel details, room availability, and customer reservations through constraints, queries, and triggers.

The React-based web app allowed clients to search for rooms using multiple criteria while employees managed reservations, stays, and payments. My favorite part was designing and implementing the SQL database. Despite a one-month timeframe and five other courses, we delivered a well-structured, functional application, earning a 97% grade.

View the project at https://github.com/glk-0/e-Hotels.
